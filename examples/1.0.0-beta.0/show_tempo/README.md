# Show Tempo

First Ableton extension test project.

The goal of this example is to read and display the current tempo of the Live Set.

## Compatibility

- Extensions SDK: 1.0.0-beta.0
- Ableton Live: 12.4.5 beta

An Ableton Live extension built with `@ableton-extensions/sdk`.
Learn about building extensions: https://ableton.github.io/extensions-sdk/
